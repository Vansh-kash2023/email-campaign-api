Personalized Email Drip Campaign API

Overview

Welcome to the Personalized Email Drip Campaign API. This API is designed to generate customized email sequences tailored to individual account personas and preferences.

Features

  • Accept account details, including industry, pain points, and campaign objectives.
  • Generate a sequence of emails for Account-Based Marketing (ABM) campaigns.
  • Provide output in JSON format for seamless integration into automation tools.

Endpoints

POST /generate_campaign

Generate a personalized email drip campaign.

Sample Input:

{
  "accounts": [
    {
      "account_name": "Acme Corp",
      "industry": "Technology",
      "pain_points": ["High operational costs", "Lack of automation"],
      "contacts": [
        {
          "name": "Jane Doe",
          "email": "jane.doe@acme.com",
          "job_title": "CTO"
        }
      ],
      "campaign_objective": "awareness"
    }
  ],
  "number_of_emails": 2
}

Sample Output:

{
  "campaigns": [
    {
      "account_name": "Acme Corp",
      "emails": [
        {
          "subject": "Reduce Your Operational Costs with Automation",
          "body": "Hi Jane, ...",
          "call_to_action": "Learn more"
        }
      ]
    }
  ]
}
