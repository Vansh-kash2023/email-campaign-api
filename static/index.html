Email Drip Campaign API

Welcome to the Email Drip Campaign API! This API allows you to create highly personalized and targeted email drip campaigns for Account-Based Marketing (ABM).

How It Works

The API leverages GenAI to generate customized email sequences tailored to individual account personas and preferences. It accepts account data, generates email sequences, and provides the output in JSON format.

API Usage

Input Contract

{
    "accounts": [
        {
            "account_name": "",
            "industry": "",
            "pain_points": ["", ""],
            "contacts": [
                {
                    "name": "",
                    "email": "",
                    "job_title": ""
                }
            ],
            "campaign_objective": ""
        }
    ],
    "number_of_emails": 
}

Output Contract

{
    "campaigns": [
        {
            "account_name": "",
            "emails": [
                {
                    "subject": "",
                    "body": "",
                    "call_to_action": ""
                }
            ]
        }
    ]
}

Example

Here is an example of how to use the API:

POST /generate-campaign/
Content-Type: application/json

{
    "accounts": [
        {
            "account_name": "Acme Corp",
            "industry": "Manufacturing",
            "pain_points": ["High production costs", "Inefficient processes"],
            "contacts": [
                {
                    "name": "John Doe",
                    "email": "john.doe@acmecorp.com",
                    "job_title": "CEO"
                }
            ],
            "campaign_objective": "awareness"
        }
    ],
    "number_of_emails": 3
}
